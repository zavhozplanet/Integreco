## ✅ РЕАЛИЗОВАНО В ТЕКУЩЕЙ СЕССИИ

### Каталог карт (2026-04-08)
- [x] **Каталог карт в главном меню:** Кнопка 📂 «Каталог карт» в ☰-меню. Читает все `.json` из рабочей папки.
- [x] **Авто-имя файла карты:** При вводе первого заголовка root-узла новой карты — файл автоматически переименовывается из `map_TIMESTAMP.json` в `LABEL.json`.
- [x] **Окно каталога:** Полноразмерный overlay с анимацией. Режимы: карточки (горизонтальные, по умолчанию) и плитки. Превью-эскизы генерируются через canvas.
- [x] **Кнопки на карточках:** 🔗 Поделиться, ⬇️ Скачать (подменю: JSON-LD/MD/PNG), 🗑️ В корзину.
- [x] **Логика открытия:** Двойной клик на карточку → новой вкладке. Если холст текущей вкладки пуст — в текущей вкладке.
- [x] **Новая вкладка через «+»:** Открывает пустой холст, карта сохраняется в рабочей папке под уникальным именем.
